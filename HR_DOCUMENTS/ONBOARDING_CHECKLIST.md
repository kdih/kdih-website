# Employee Onboarding Checklist

**Katsina Digital Innovation Hub**

This checklist ensures a smooth onboarding process for all new employees.

---

## PRE-ARRIVAL (Before Day 1)

### HR/Admin Tasks

- [ ] Offer letter signed and returned by candidate
- [ ] Employment contract prepared and sent for review
- [ ] Background/reference checks completed (if applicable)
- [ ] Employee file created (physical and digital)
- [ ] Email account created ([firstname.lastname]@kdih.org)
- [ ] User added to relevant systems and platforms
- [ ] Desk/workspace prepared and assigned
- [ ] Equipment prepared (computer, accessories, etc.)
- [ ] Access card/keys arranged
- [ ] Welcome email sent with first-day instructions

### IT/Technical Tasks

- [ ] Computer/laptop assigned and configured
- [ ] Software installed (MS Office, necessary tools)
- [ ] Network access credentials created
- [ ] Email account tested and functional
- [ ] Phone extension assigned (if applicable)
- [ ] Access to shared drives and folders configured

### Supervisor/Team Tasks

- [ ] First-day schedule prepared
- [ ] Buddy/mentor assigned
- [ ] Workspace tour planned
- [ ] Team introduction meeting scheduled
- [ ] First week training plan outlined

---

## DAY 1 - First Day

### Morning (9:00 AM - 12:00 PM)

**Reception/Welcome (9:00 AM - 9:30 AM)**

- [ ] Warmly welcomed by front desk
- [ ] Visitor pass/temporary access provided
- [ ] Directed to meeting point

**HR Orientation (9:30 AM - 11:00 AM)**

- [ ] Signed employment contract collected
- [ ] Employee information form completed
- [ ] Required documents submitted and verified:
  - [ ] National ID / International Passport / Driver's License
  - [ ] Academic certificates and transcripts
  - [ ] Professional certificates
  - [ ] Previous employment references  
  - [ ] Passport photographs
  - [ ] Bank account details
  - [ ] Tax Identification Number (TIN)
- [ ] Employee handbook provided and reviewed
- [ ] Code of conduct explained
- [ ] Organizational policies reviewed:
  - [ ] Working hours and attendance
  - [ ] Leave policy
  - [ ] Dress code
  - [ ] Health and safety
  - [ ] Confidentiality
- [ ] Benefits package explained
- [ ] Payroll and tax information collected
- [ ] Emergency contact information recorded
- [ ] Q&A session

**Facility Tour (11:00 AM - 11:30 AM)**

- [ ] Office layout and workspaces
- [ ] Training rooms
- [ ] Coworking space
- [ ] Meeting rooms
- [ ] Restrooms and facilities
- [ ] Kitchen/break room
- [ ] Emergency exits and assembly points
- [ ] Parking area
- [ ] Security protocols explained

**Equipment Setup (11:30 AM - 12:00 PM)**

- [ ] Workstation/desk shown
- [ ] Computer/laptop handed over
- [ ] Login credentials provided
- [ ] Phone/extension set up
- [ ] Access card/keys issued
- [ ] Stationery provided

### Lunch Break (12:00 PM - 1:00 PM)

- [ ] Lunch with team or buddy

### Afternoon (1:00 PM - 5:00 PM)

**Department Orientation (1:00 PM - 2:00 PM)**

- [ ] Introduction to supervisor
- [ ] Team meeting and introductions
- [ ] Department overview
- [ ] Role-specific expectations discussed
- [ ] First week priorities outlined

**Systems & Tools Training (2:00 PM - 4:00 PM)**

- [ ] Email setup and demonstration
- [ ] Shared drives and folders access
- [ ] KDIH website and member portal overview
- [ ] Task/project management tools
- [ ] Communication tools (WhatsApp groups, etc.)
- [ ] Time/attendance tracking system
- [ ] Any role-specific software

**Administrative Tasks (4:00 PM - 4:30 PM)**

- [ ] IT acceptable use policy signed
- [ ] Confidentiality agreement signed
- [ ] Photo taken for ID card
- [ ] Added to organizational chart
- [ ] Added to email distribution lists
- [ ] Emergency contact form submitted

**Day 1 Wrap-Up (4:30 PM - 5:00 PM)**

- [ ] Questions answered
- [ ] Day 2 expectations reviewed
- [ ] Feedback on Day 1 experience collected
- [ ] Welcome gift presented (if applicable)

---

## WEEK 1 - First Week

### Day 2-5

**Daily Check-ins**

- [ ] Morning check-in with supervisor (15 min)
- [ ] End-of-day check-in with buddy/mentor (15 min)

**Training & Learning**

- [ ] KDIH mission, vision, and values deep-dive
- [ ] Organizational structure and key personnel
- [ ] Programs and services overview
  - [ ] Training programs
  - [ ] Coworking services
  - [ ] Startup incubation
  - [ ] Consulting services
- [ ] Partner organizations and stakeholders
- [ ] Current projects and initiatives
- [ ] Role-specific training sessions
- [ ] Shadow team members in related roles

**Hands-On Activities**

- [ ] Attend team meetings
- [ ] Observe department workflows
- [ ] Assigned initial tasks (low-stakes, supervised)
- [ ] Begin familiarization with key processes

**Administrative**

- [ ] Complete all outstanding paperwork
- [ ] Permanent ID card issued
- [ ] Added to all relevant communication channels
- [ ] Performance objectives discussed and documented

**End of Week 1**

- [ ] Week 1 review meeting with supervisor
- [ ] Feedback provided and concerns addressed
- [ ] Week 2 goals set
- [ ] Employee feedback survey completed

---

## WEEK 2-4 - First Month

### Week 2

- [ ] Increased task responsibilities
- [ ] Attend relevant meetings independently
- [ ] Meet individually with key stakeholders
- [ ] Deeper training on role-specific tools and processes
- [ ] Begin contributing to team projects
- [ ] Submit first attendance/timesheet (if applicable)

### Week 3

- [ ] Take ownership of assigned tasks
- [ ] Start independent work with supervisor oversight
- [ ] Participate actively in team activities
- [ ] Meet with HR for check-in
- [ ] Address any challenges or questions
- [ ] Cultural and team integration activities

### Week 4

- [ ] Fully integrated into team workflows
- [ ] Handling regular job responsibilities
- [ ] Mid-probation review (if 3-month probation)
- [ ] Salary processing confirmed for first month
- [ ] Performance goals reviewed and adjusted if needed

**End of Month 1**

- [ ] Month 1 performance review with supervisor
- [ ] Employee completes probation feedback form
- [ ] Identify any training needs
- [ ] Confirm understanding of role expectations

---

## MONTH 2-3 - Probation Period

### Ongoing Activities

- [ ] Weekly 1-on-1 meetings with supervisor
- [ ] Regular performance feedback
- [ ] Progressive independence in tasks
- [ ] Building relationships across teams
- [ ] Participating in KDIH events and programs
- [ ] Contributing ideas and suggestions

### Month 2

- [ ] Take on more complex projects
- [ ] Represent department in meetings (if applicable)
- [ ] Month 2 performance review

### Month 3

- [ ] Probation review meeting scheduled
- [ ] Performance evaluation completed
- [ ] Employee self-assessment submitted
- [ ] Feedback from colleagues gathered (360 review)
- [ ] Decision on confirmation communicated
- [ ] If confirmed: permanent status letter issued
- [ ] If extended: feedback and development plan provided

**End of Probation**

- [ ] Formal probation completion meeting
- [ ] Confirmation letter issued (if successful)
- [ ] Celebrate and welcome fully to the team!
- [ ] Discuss long-term goals and career development

---

## POST-PROBATION

### Month 6 (If 6-month probation)

- [ ] 6-month performance review
- [ ] Probation completion decision
- [ ] Confirmation letter issued (if successful)
- [ ] Benefits activation (health insurance, etc.)
- [ ] Long-term goal setting

---

## ONGOING SUPPORT

### Continuous Integration

- [ ] Monthly check-ins with supervisor (ongoing)
- [ ] Quarterly performance reviews (ongoing)
- [ ] Access to professional development opportunities
- [ ] Involvement in team-building activities
- [ ] Regular feedback and recognition
