**Common Placeholders:**

- `[Position Title]` - Exact job title
- `[Date]` - Relevant date
- `[Amount]` - Salary or compensation amount
- `[Number]` - Numerical values (leave days, probation months)

**Examples:**

- `[3/6]` - Choose either 3 or 6 (for probation period)

## üí° Best Practices

### Recruitment

- Follow the same process for all candidates
- Use the interview guide consistently
- Score all candidates using the evaluation form
- Keep records of all interview notes

### Onboarding

- Start preparation before Day 1
- Assign a buddy/mentor for each new employee
- Complete all checklist items
- Collect feedback throughout probation

### Employment Contracts

- Review all terms with the employee
- Allow time for questions
- Keep signed copies on file
- Provide employee with their copy
